赛博朋克诞生于 1980 年代的文学运动，但它的美学影响力远远超出了文学范畴，深刻影响了电影、游戏、时尚和 UI 设计。

## 文学起源

威廉·吉布森的《神经漫游者》（Neuromancer）定义了赛博朋克的视觉语言：「天空像调错频道的电视屏幕，灰白而沉寂。」Bruce Sterling、Philip K. Dick 等作家的作品共同构建了这个「高科技、低生活」的反乌托邦世界。

## 视觉符号系统

霓虹灯、全息投影、数据流、植入体、义肢、雨夜、摩天楼与贫民窟的并置——这些视觉元素构成了赛博朋克的符号系统。在 UI 设计中，这些元素被抽象为发光边框、半透明面板、数据可视化、扫描线纹理等。

## UI 设计原则

`High Tech, Low Life` — 用尖端的技术呈现破败的美感。 `信息过载` — 屏幕上同时展示大量数据，制造沉浸感。 `功能优先` — 每个视觉元素都有其功能含义，而非纯粹装饰。 `对比强烈` — 黑暗背景与霓虹亮色的极端对比。

## 现代 Web 中的赛博朋克

近年来赛博朋克美学在 Web 设计中复兴，从 GitHub 的暗色主题到各种开发者工具，深色背景 + 语法高亮的组合本身就是一种赛博朋克审美的体现。这种风格特别适合技术人员的个人品牌——它传达了一种黑客精神和技术自信。
