一直以来，我都想拥有一个与众不同的个人网站——不是那种千篇一律的极简白底黑字，而是一个真正能代表我审美和技术热情的赛博朋克数字空间。

## 设计灵感的来源

赛博朋克（Cyberpunk）不仅仅是霓虹灯和黑暗街道，它是一种反叛精神，一种对科技与人性的深度思考。在视觉层面，我们从《银翼杀手》《攻壳机动队》《赛博朋克2077》等经典作品中汲取灵感。

核心设计元素包括：深色背景营造沉浸感、霓虹色点缀产生视觉焦点、等宽字体传达技术感、扫描线叠加增加复古 CRT 质感。

## 技术选型：为什么不用框架？

在这个 React/Vue 横行的时代，选择纯原生技术栈似乎是一种倒退。但实际上，对于一个个人博客而言，零框架意味着：

`极致性能` — 没有运行时开销，首屏加载在 100ms 以内。 `零依赖` — 不担心 npm 供应链安全，不担心版本升级。 `长久可维护` — 十年后依然能跑，HTML/CSS/JS 永不过时。

## 色彩系统的设计

一个好的色彩系统是赛博朋克设计的基石。我选择了深空黑（#0a0a0f）作为主背景色，它比纯黑更有层次感。霓虹绿色（#00ff41）作为主强调色，让人联想到经典终端和 Matrix 数字雨。同时通过 CSS Custom Properties 实现了主题切换功能——一键在霓虹绿、青色、品红之间切换。

## CRT 扫描线效果

使用 `repeating-linear-gradient` 配合 `pointer-events: none` 的伪元素，仅用 6 行 CSS 就实现了逼真的 CRT 扫描线纹理。

## Glitch 故障文字

利用 `::before` 和 `::after` 伪元素，分别裁剪文字的不同区域并施加微小的位移动画，配合青和品红的颜色偏移，实现了经典的故障艺术效果。
